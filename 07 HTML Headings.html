The element is placed between the tag and the tag: Example My First HTML . . . Note: Metadata typically define the document title, character set, styles, scripts, and other meta information. How to View HTML Source? Have you ever seen a Web page and wondered "Hey! How did they do that?" View HTML Source Code: Right-click in an HTML page and select "View Page Source" (in Chrome) or "View Source" (in Edge), or similar in other browsers. This will open a window containing the HTML source code of the page. Inspect an HTML Element: Right-click on an element (or a blank area), and choose "Inspect" or "Inspect Element" to see what elements are made up of (you will see both the HTML and the CSS). You can also edit the HTML or CSS on-the-fly in the Elements or Styles panel that opens.
